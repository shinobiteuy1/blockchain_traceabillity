# Phase 5 — Frontend Applications

## Current Prototype

This phase contains a lightweight UX prototype for the infrastructure workflow.

## Goals

- help non-blockchain users create a local Fabric network setup
- provide guided organization and member workflows
- connect the UI to the backend prototype
- demonstrate how infrastructure actions map to the local Fabric bootstrap

## Run locally

1. Start the backend container or backend process.
2. Open `http://localhost:5000/ui/index.html`.

## Demo flow

1. Create a network
2. Create an organization
3. Add members to the organization
4. Review the generated Fabric readiness plan
